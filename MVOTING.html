    투표 만들기

    투표 카테고리
    투표 유형을 설정 할 수 있습니다.
    정책투표 : 가장 일반적인 투표로 객관식,주관식 또는 복합투표가 가능합니다.
    VS 형 : 간단한 예/아니오 형식의 투표로 사용자가 간편하게 투표할 수 있습니다.
    다중투표 : 성격이 다른 정책투표 2개를 묶은 특수투표입니다. 이용시 문의바랍니다.
    분기투표 : 설문조사 형식으로 사용되며 사용자가 선택한 답변에 따라 다음 질문이 변경됩니다.

    정책투표 카테고리
    만드는 투표가 정책적으로 반영예정인지 참고사항인지를 선택합니다.
    정책반영투표 : 투표의 결과가 실제 정책에 반영되는 경우에 설정하는 투표로 결과 노출, 투표대상, 통계정보 등 일부 설정값이 고정됩니다.
    정책참고투표 : 투표의 결과가 실제 정책과 관계없이 참고용인 투표입니다.

    투표 이미지 등록
    투표의 대표 이미지를 등록합니다.
    투표의 대표이미지 및 추가 이미지를 등록가능하며 용량제한은 이미지 1개당 2MB 입니다.
    이미지는 JPG,JPEG,PNG 포맷을 지원하며 정사각형 또는 4X3 비율의 가로형의 직사각형 이미지를 추천드립니다.

    투표 제목
    투표의 제목을 입력합니다. 글자수 제한은 없지만 리스트에 표출되는 영역의 한계가 있기 때문에 20자 이내를 추천합니다.

    투표 표시 유형
    투표의 질문영역의 표현 스타일을 정합니다.
    일반 : 질문이 세로로 모두 표출됩니다.
    블록 : 모든 질문이 상단에 블록형식으로 표현되며 선택된 질문만 리스트로 표출됩니다.
    		질문 순서 랜덤
    입력한 모든 질문영역이 접속할때마다 랜덤으로 표출됩니다.
    		총 투표수 설정
    각 질문별 답변의 선택수가 아닌 모든 질문의 투표수를 설정합니다.
    *특수한경우에만 사용되며 일반적으로 사용되지 않습니다.
    해당기능에 대해 문의사항이 있으신분은 담당자에게 연락 부탁드립니다.

    추가 사항

    투표 취지
    투표에 대한 간단한 취지를 입력 할 수 있습니다. 일반적으로 투표의 기간 및 대상, 투표를 진행하는 부서 등의 정보를 입력합니다.
    자세히보기 URL
    투표에 대한 관련 사이트가 있다면 입력합니다. 사용자 화면에서 버튼으로 표현되며 새창으로 입력된 URL을 표출합니다.
    투표 기간
    투표의 기간을 설정하며 "분"단위까지 설정합니다.
    투표의 기본 설정값은 접속당일 오전 09:00 ~ 일주일 후 오전 09:00 입니다.
    시작일
    종료일
    투표 대상
    [안내] 투표대상에 대한 설정을 합니다.
    전체공개(휴대폰) : 가장 일반적인 투표이며 휴대폰번호 인증으로만 투표를 진행 가능합니다.
                                  ※ 휴대폰이 3개일 경우 3표 행사 가능
    전체공개(소셜, 휴대폰) : 휴대폰번호,소셜인증으로 투표를 받습니다.
                                  ※ 휴대폰이 3개일 경우 + 소셜 아이디 3개일 경우 6표 행사 가능
    전체공개(통합인증 - 시민카드) : 서울시 본인인증으로 투표합니다. 1인 1투표가 가능합니다.
                                  ※ 시민카드 앱 설치 및 회원가입->인증서 발급 이 필요합니다.
    전체공개(통합인증 - 시민카드,나이스) : 서울시 본인인증 및 나이스 본인인증으로 투표합니다. 1인 1투표가 가능합니다.
                                 ※ 시민카드설치가 부담스러운경우 나이스를 병행하여 투표가 가능합니다.나이스 인증은 시민 대부분 설치되어 있습니다.
    특정인 : 전화번호 명부를 업로드하거나 지역정보, 위치정보를 선택하여 해당 조건을 충족하는 특정인을 대상으로 투표를 진행할 수 있습니다.
    비공개 투표(휴대폰) : 비밀번호를 설정하여 비밀번호를 아는 경우에만 휴대폰 인증으로 투표 할 수 있습니다.
    비공개 투표(소셜, 휴대폰) : 비밀번호를 설정하여 비밀번호를 아는 경우에만 휴대폰 인증또는 소셜 아이디로 투표 할 수 있습니다.
    "선택사항이 팝업창으로 표시됩니다. 팝업 차단을 해제하여 주십시오."
    선착순 투표 설정
    설정한 인원이 투표의 참여 시 투표가 자동종료 됩니다.
    투표의 표시범위
    API 기능 사용시에만 노출되며 등록되는 투표가 엠보팅, 또는 본인의 API 사이트에 노출되는 여부를 선택 합니다.
    전화번호 차단
    해당투표에 투표하지 못하는 휴대폰 번호를 업로드 합니다.
    휴대폰 인증 투표에만 사용 가능하며 꼭! 양식받기를 하여 양식에 맞는 휴대폰 번호를 업로드 해주세요
    * 엑셀( *.xls ) 확장자만 지원 , (-,/) 등 특수문자는 입력 불가능
    예) 01000000000 ( 0 ) , 010-0000-0000 ( X )
    재 투표 가능 여부
    투표 종료 시 까지 여러 번 투표가 가능하게 설정합니다. 저장되는 투표는 마지막 투표만 저장되며 마지막 투표만 인정됩니다.

    기타 옵션

    결과노출
    투표의 결과노출 설정입니다.
    참여시 결과노출 : 투표 참여 시 투표 참여자는 실시간 득표수 확인이 가능합니다.
    마감후 결과노출 : 투표 마감 이후에 모든 시민이 투표결과를 확인할 수 있습니다.
    결과노출 안함 : 투표 마감 이후에도 관리자를 제외한 시민들은 투표결과를 확인할 수 없습니다.
    투표발제자
    투표상에 노출되는 투표 발제자의 이름입니다. 일반적으로 부서명 또는 기관명을 입력 합니다.
    		등록자
    통계 정보
    사용자가 직접 입력하는 [생년],[성별],[지역] 정보로, 받도록 설정할 경우 정보를 입력하지 않으면 투표에 참여할 수 없습니다.
    댓글
    사용자가 댓글을 작성할 수 있는지에 대한 설정을 합니다. "받지않음" 선택시 댓글란은 비노출 됩니다.

    태그 입력
    투표를 표현하는 태그를 입력해 주세요!
    입력한 태그는 엠보팅 알고리즘으로 사용자에게 맞춤형 투표로 추천됩니다.
